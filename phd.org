* Сведения из теории линейных кодов
** Определения
Поле \( \mathbb{F}_q \), векторы \( \vect{v}\in\mathbb{F}^n_q \), матрицы \( \matr{M}\in\mathbb{F}^{k\times n}_q \), группа \( \str{S}_n \), элемент \( \elt{\sigma} \), код \( \code{C} \), алгоритм \( \algo{Decode} \), сложность \( \compl{O}(n^2) \), шифр \( \crypto{E} \).

#+name: def:nkcode
#+begin_definition
\( [n]_q \)-код \( \code{C} \) называется \( [n,k]_q \)-кодом, если \( \dim\code{C}=k \).
#+end_definition
#+name: def:generator
#+begin_definition
Матрица \( \matr{G}\in\mathbb{F}^{k\times n}_q \), строки --- базис \( \code{C} \), называется /порождающей/.
#+end_definition
#+name: thm:dual
#+begin_theorem
\( \code{C}^\perp \)~--- \( [n,n{-}k]_q \)-код и \( (\code{C}^\perp)^\perp=\code{C} \).
#+end_theorem
#+begin_proof
Из теории линейных пространств.
#+end_proof
#+begin_corollary
\( \matr{G}=(\matr{I}_k\mid\matr{G}') \Rightarrow \matr{H}=(-\matr{G}'^\top\mid\matr{I}_{n-k}) \).
#+end_corollary
#+name: lem:weight
#+begin_lemma
\( d(\code{C})=\min_{\vect{c}\neq\vect{0}}\operatorname{wt}(\vect{c}) \).
#+end_lemma
#+begin_proposition
Число порождающих матриц: \( (q^k{-}1)(q^k{-}q)\cdots(q^k{-}q^{k-1}) \).
#+end_proposition
#+begin_remark
Не у каждого кода есть систематическая матрица.
#+end_remark
#+begin_example
Код Хемминга \( [7,4,3]_2 \) исправляет одну ошибку.
#+end_example
#+begin_problem
Описать самодуальные \( [n,n/2]_2 \)-коды при \( n\leq 24 \).
#+end_problem

** Кодовое расстояние и спектр
#+begin_definition
Минимальное расстояние: \( d=\min_{\vect{c}\neq\vect{0}}\operatorname{wt}(\vect{c}) \).
#+end_definition
#+begin_definition
Спектр весов: \( \vect{A}(\code{C})=(A_0,\ldots,A_n) \), \( A_i=|\{\vect{c}\mid\operatorname{wt}(\vect{c})=i\}| \).
#+end_definition

* Алгоритмы и листинги
#+caption: Спектр весов
#+name: alg:spectrum
\begin{algorithm}[H]
\KwInput{$\matr{G}\in\mathbb{F}^{k\times n}_2$}
\KwOutput{спектр $\vect{A}$}
\For{$\vect{a}\in\mathbb{F}^k_2$}{
    $\vect{c}\gets\vect{a}\matr{G}$\tcp*{кодовое слово}
    $w\gets\operatorname{wt}(\vect{c})$\;
    \Comment{обновляем}
    \uIf{$w=0$}{$A_0{+}{+}$\;}
    \uElseIf{$w=n$}{$A_n{+}{+}$\;}
    \Else{$A_w{+}{+}$\;}
}
\Return{$\vect{A}$}\;
\end{algorithm}
Ссылка: алгоритм\nbsp{}\ref{alg:spectrum}.

\begin{lstlisting}[language=Python,caption={Python},label={lst:py}]
import numpy as np
def weight_spectrum(G):
    """Спектр весов двоичного кода."""
    k, n = G.shape
    A = np.zeros(n+1, dtype=int)
    for i in range(2**k):
        a = np.array([(i>>j)&1 for j in range(k)])
        c = a @ G % 2  # Кодовое слово
        A[np.sum(c)] += 1
    return A
print("Спектр:", weight_spectrum(np.array([[1,0,1,1],[0,1,1,0]])))
\end{lstlisting}

\begin{lstlisting}[language=C,caption={C},label={lst:c}]
#include <stdio.h>
/* Вес Хемминга */
int wt(unsigned x) {
    int w=0;
    // Считаем биты
    while(x){w+=x&1;x>>=1;}
    return w;
}
int main(void){printf("wt(0xB)=%d\n",wt(0xB));return 0;}
\end{lstlisting}

(*@\lstInput@*): $\vect{y}$, $\matr{H}$
(*@\lstOutput@*): ближайшее кодовое слово
procedure Decode($\vect{y}$, $\matr{H}$)
begin
    $\vect{s}\gets\matr{H}\vect{y}^\top$
    if $\vect{s}=\vect{0}$ then return $\vect{y}$
    for $j=1$ to $n$ do
        if $\vect{s}$ = столбец $j$ then return $\vect{y}\oplus\vect{e}_j$
    return nil
end
\end{lstlisting}

\begin{lstpseudocode}[caption={Произведение Шура},label={alg:schur}]
(*@\lstInput@*): базисы $\code{C}_1$, $\code{C}_2$
(*@\lstOutput@*): базис $\code{C}_1\star\code{C}_2$
function SchurProduct($\code{C}_1$, $\code{C}_2$)
begin
    $B\gets\emptyset$
    for each $\vect{c}_1,\vect{c}_2$ do
        $B\gets B\cup\{\vect{c}_1\star\vect{c}_2\}$
    return RowReduce($B$)
end
\end{lstpseudocode}
