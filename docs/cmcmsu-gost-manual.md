# cmcmsu-gost.sty — Руководство пользователя

**Версия:** 0.3 от 21 марта 2026 г.

Пакет `cmcmsu-gost` обеспечивает оформление документов LaTeX в соответствии с требованиями ГОСТ (в частности, ГОСТ Р 7.0.11-2011). Предназначен для использования с классами `article` и `report`.

---

## 1. Подключение пакета

```latex
\usepackage{cmcmsu-gost}
```

Или с опциями:

```latex
\usepackage[spacing=15, tocpagewordon]{cmcmsu-gost}
```

### Совместимость с движками

Пакет поддерживает **pdfLaTeX**, **XeLaTeX** и **LuaLaTeX**. Выбор кодировки и шрифтового механизма происходит автоматически:

- **pdfLaTeX** — загружаются `inputenc` (utf8) и `fontenc` (T1, T2A), если пользователь не подключил их самостоятельно.
- **XeLaTeX / LuaLaTeX** — загружается `fontspec`, если не подключён заранее.

Если ни `babel`, ни `polyglossia` не загружены пользователем, пакет подключит `babel` с опцией `russian`.

> **Совет.** Если вы используете пакет `cmcmsu-gost-fontselector` для настройки шрифтов, кодировки и математические пакеты будут загружены им, и `cmcmsu-gost` не станет их дублировать.

## 2. Опции пакета

Все опции можно передать при подключении пакета либо через команду `\cmcmsugostsetup` (см. раздел 3).

### 2.1. `spacing` — межстрочный интервал

| Значение | Результат                  |
|----------|----------------------------|
| `10`     | Одинарный интервал (1.0)   |
| `15`     | Полуторный интервал (1.5)  |
| `20`     | Двойной интервал (2.0)     |

**По умолчанию:** `15` (полуторный).

```latex
\usepackage[spacing=15]{cmcmsu-gost}   % полуторный
\usepackage[spacing=20]{cmcmsu-gost}   % двойной
```

### 2.2. `tocpageword` — слово «Стр.» в оглавлении

Если включена, над колонкой номеров страниц в оглавлении печатается заголовок **«Стр.»**, который автоматически повторяется на каждой странице оглавления.

| Форма                     | Действие    |
|---------------------------|-------------|
| `tocpageword=true`        | Включить    |
| `tocpageword=false`       | Выключить   |
| `tocpagewordon` (краткая) | Включить    |
| `tocpagewordoff` (краткая)| Выключить   |

**По умолчанию:** выключено.

```latex
\usepackage[tocpagewordon]{cmcmsu-gost}
```

### 2.3. `sectionclearpage` — разделы с новой страницы

Каждый `\section` будет начинаться с новой страницы. **Действует только для класса `article`.** В классе `report` главы (`\chapter`) и так начинаются с новой страницы.

| Форма                          | Действие    |
|--------------------------------|-------------|
| `sectionclearpage=true`        | Включить    |
| `sectionclearpage=false`       | Выключить   |
| `sectionclearpageon` (краткая) | Включить    |
| `sectionclearpageoff` (краткая)| Выключить   |

**По умолчанию:** выключено.

```latex
\usepackage[sectionclearpageon]{cmcmsu-gost}
```

## 3. Команда `\cmcmsugostsetup`

Альтернативный способ задать опции **в преамбуле** (до `\begin{document}`):

```latex
\usepackage{cmcmsu-gost}
\cmcmsugostsetup{
    spacing         = 15,
    tocpageword     = true,
    sectionclearpage = false
}
```

> **Важно.** Вызов `\cmcmsugostsetup` после `\begin{document}` вызовет ошибку компиляции.

## 4. Геометрия страницы

Пакет устанавливает следующие параметры страницы (A4):

| Параметр        | Значение |
|-----------------|----------|
| Верхнее поле    | 2 см     |
| Нижнее поле     | 2 см     |
| Левое поле      | 2,5 см   |
| Правое поле     | 1 см     |
| Маргинальные заметки | отключены |

Абзацный отступ установлен в `2.5em` (≈ 5 знаков), как требует ГОСТ Р 7.0.11-2011, п. 5.3.7.

---

## 5. Колонтитулы и нумерация страниц

- Номер страницы — **по центру верхнего колонтитула**.
- Линейка под колонтитулом **отсутствует**.
- В классе `report` страницы глав используют тот же стиль (стандартный `plain` заменён на `fancy`).
- Номер титульной страницы учитывается в сквозной нумерации.

---

## 6. Заголовки разделов

Все уровни заголовков (`\chapter`, `\section`, `\subsection`, `\subsubsection`) оформляются единообразно:

- **Шрифт:** 14 pt, полужирный.
- **Выравнивание:** по левому краю.
- **Отступ слева:** 0 pt.
- **Вертикальный интервал** до и после заголовка: 3 базовых строки.

Для класса `report` главы форматируются с префиксом «Глава N.»:

```
Глава 1. Название главы
```

---

## 7. Оглавление

- Заголовок оглавления — **«Оглавление»** (переопределяется через `\captionsrussian` в `babel`).
- Все уровни (включая главы) используют **точечные лидеры** до номера страницы.
- Номера страниц набраны **прямым** (не полужирным) шрифтом.
- Номера разделов завершаются точкой: «1. Название», «1.1. Название».
- В классе `report` перед номером главы в оглавлении стоит слово «Глава».
- После вывода оглавления автоматически вставляется разрыв страницы.

## 8. Списки

### Маркированные списки (`itemize`)

Маркер на всех уровнях вложенности — **тире** (`--`), полужирное.

### Нумерованные списки (`enumerate`)

- Второй уровень нумерации (`enumii`) использует **русские строчные буквы**: а), б), в), …
- Третий уровень (`enumiii`) — **арабские цифры** с закрывающей скобкой: 1), 2), 3), …

### Общие параметры

Для всех списков:
- Вертикальные промежутки между элементами убраны (`nosep`).
- Метка выровнена относительно абзацного отступа (`labelindent=\parindent`).

---

## 9. Рисунки

- Нумерация **внутри раздела** (или главы): Рисунок 2.1, Рисунок 2.2, …
- Разделитель между меткой и названием — **тире**: `Рисунок 1.1 --- Название`.
- Подпись — **под рисунком**, центрирована для коротких подписей.

### Субрисунки

- Поддерживаются подтипы (`subfigure`).
- Нумерация субрисунков — **русскими строчными буквами**: а), б), в), …

---

## 10. Таблицы

- Нумерация **внутри раздела** (или главы): Таблица 1.1, Таблица 1.2, …
- Разделитель — **тире**: `Таблица 1.1 --- Название`.
- Подпись — **над таблицей**, выровнена по ширине.

---

## 11. Формулы

- Нумерация формул — **внутри раздела** (или главы): (1.1), (1.2), …
- В окружении `subequations` подформулы нумеруются **русскими буквами**: (1.1а), (1.1б), …
- Вертикальные отступы вокруг выключных формул скорректированы для компактности.

---

## 12. Приложения

После команды `\appendix` разделы (или главы) автоматически оформляются как приложения:

```
Приложение А Название
Приложение Б Название
```

- Нумерация — **русскими прописными буквами** (А, Б, В, Г, …).
- В оглавлении отображаются с точкой: «Приложение А. Название».
- Для `report` переопределяется `\chapter`, для `article` — `\section`.

---

## 13. Загружаемые пакеты

Пакет автоматически загружает перечисленные ниже пакеты. Если вам нужно передать им специальные опции, подключите нужный пакет **до** `cmcmsu-gost`.

**Графика и таблицы:** `graphicx`, `booktabs`, `longtable`, `tabularx`, `multirow`, `makecell`, `colortbl`, `wrapfig`, `rotating`.

**Подписи:** `caption`, `capt-of`, `chngcntr`.

**Разметка:** `geometry`, `fancyhdr`, `titlesec`, `tocloft`, `enumitem`, `indentfirst`.

**Набор:** `setspace`, `hyphenat`, `ulem` (режим `normalem`).

**Ссылки:** `hyperref` (с опцией `linktoc=page`).

**Служебные:** `iftex`, `kvoptions`, `etoolbox`, `afterpage`.

---

## 14. Обработка переполнений строк

Пакет настраивает параметры разрыва строк для минимизации предупреждений `Overfull \hbox`:

- `\tolerance=1414`, `\emergencystretch=1.5em`, `\sloppy`.
- Запрещены висячие строки (`\clubpenalty=10000`, `\widowpenalty=10000`).
- Включён `\raggedbottom`.

---

## 15. Минимальный пример

### article (курсовая, статья)

```latex
\documentclass[14pt]{extarticle}
\usepackage{cmcmsu-gost}

\begin{document}

\tableofcontents

\section{Введение}
Текст введения.

\section{Основная часть}
\subsection{Подраздел}
Текст подраздела.

\appendix
\section{Листинги кода}
Код программы.

\end{document}
```

### report (дипломная работа)

```latex
\documentclass[14pt]{extreport}
\usepackage[tocpagewordon, spacing=15]{cmcmsu-gost}

\begin{document}

\tableofcontents

\chapter{Введение}
Текст введения.

\chapter{Обзор литературы}
Текст главы.

\appendix
\chapter{Дополнительные материалы}
Содержимое приложения.

\end{document}
```

## 16. Краткая справка

| Что настраивается          | Как задать                                      |
|----------------------------|-------------------------------------------------|
| Межстрочный интервал       | `spacing=10` / `15` / `20`                      |
| «Стр.» в оглавлении       | `tocpageword=true` или `tocpagewordon`           |
| Разделы с новой страницы   | `sectionclearpage=true` или `sectionclearpageon` |
| Настройка в преамбуле      | `\cmcmsugostsetup{ключ=значение, …}`            |
