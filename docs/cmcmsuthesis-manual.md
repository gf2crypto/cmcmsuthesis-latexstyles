# cmcmsuthesis.cls — Руководство пользователя

**Версия:** 0.1 от 23 марта 2026 г.

Класс `cmcmsuthesis` — базовый класс для оформления выпускных квалификационных работ, курсовых и диссертаций факультета ВМК МГУ. Класс автоматически загружает все три пакета комплекта (`cmcmsugost-fontselector`, `cmcmsu-gost`, `cmcmsugost-extra`) и предоставляет готовую титульную страницу с настраиваемыми метаданными.

---

## 1. Подключение

```latex
\documentclass{cmcmsuthesis}
```

Или с опциями:

```latex
\documentclass[font=libertinus, titlepagesty=gost, draft]{cmcmsuthesis}
```

Класс наследует от `extarticle` (по умолчанию) или `extreport` и автоматически подключает:

- `cmcmsugost-fontselector` — шрифты и математика
- `cmcmsu-gost` — оформление по ГОСТ (с опциями `tocpageword=true`, `sectionclearpage=true`)
- `cmcmsugost-extra` — теоремы, алгоритмы, листинги, библиография, русские операторы

## 2. Опции класса

### 2.1. `parent` — базовый класс

Определяет родительский класс LaTeX.

| Значение | Результат |
|---|---|
| `extarticle` | Статья без глав (курсовая, бакалаврская работа) |
| `extreport` | Отчёт с главами (дипломная работа, диссертация) |

**По умолчанию:** `extarticle`.

```latex
\documentclass[parent=extreport]{cmcmsuthesis}
```

### 2.2. `fontsize` — размер основного шрифта

Передаётся родительскому классу.

**По умолчанию:** `14pt`.

```latex
\documentclass[fontsize=12pt]{cmcmsuthesis}
```

### 2.3. `font` — набор шрифтов

Передаётся в `cmcmsugost-fontselector` как опция `font`. Принимает любое имя набора или отдельного шрифта, описанное в документации fontselector.

**По умолчанию:** `modern`.

```latex
\documentclass[font=libertinus]{cmcmsuthesis}
\documentclass[font=pt]{cmcmsuthesis}
\documentclass[font=ebgaramond]{cmcmsuthesis}
```

### 2.4. `titlepagesty` — стиль титульной страницы

| Значение | Описание |
|---|---|
| `cmc` | Стиль ВМК: логотип сверху, университет/факультет/кафедра, подзаголовок прописными буквами |
| `gost` | Диссертационный стиль по ГОСТ Р 7.0.11-2011: логотип слева, «На правах рукописи» справа, специальность |

**По умолчанию:** `cmc`.

```latex
\documentclass[titlepagesty=gost]{cmcmsuthesis}
```

### 2.5. `draft` — черновой режим

Включает черновой режим родительского класса (рамки вместо изображений, маркеры переполнения) и отключает `microtype`.

**По умолчанию:** `false`.

```latex
\documentclass[draft]{cmcmsuthesis}
```

## 3. Метаданные работы

Метаданные задаются через систему ключей `pgfkeys`. Есть три способа их указать.

### 3.1. Стандартные команды LaTeX

```latex
\author{Иванов Иван Иванович}
\title{Разработка алгоритма оптимизации нейронных сетей}
```

### 3.2. Команда `\thesissetup`

Универсальная команда для задания любых полей:

```latex
\thesissetup{
    university  = {Московский государственный университет
                   имени М.В.\,Ломоносова},
    faculty     = {Факультет вычислительной математики
                   и кибернетики},
    department  = {Кафедра информационной безопасности},
    subtitle    = {Выпускная квалификационная работа},
    state       = Москва,
    copyright   = {На правах рукописи},
    logo        = {cmcmsuthesis-cmc}
}
```

### 3.3. Специализированные команды

```latex
\supervisor{
    name   = {канд.~физ.-мат.~наук, доцент Петров~П.П.},
    prefix = {Научный руководитель}
}

\specialty{
    code   = {1.2.3},
    name   = {теоретическая информатика, кибернетика},
    prefix = {Специальность}
}

\subtitle{Выпускная квалификационная работа}
```

## 4. Полный справочник метаданных

### 4.1. Общие поля (`\thesissetup{...}`)

| Ключ | Умолчание | Описание |
|---|---|---|
| `university` | Московский государственный университет имени М.В. Ломоносова | Название вуза |
| `faculty` | Факультет вычислительной математики и кибернетики | Название факультета |
| `department` | Кафедра информационной безопасности | Название кафедры |
| `subtitle` | Выпускная квалификационная работа | Подзаголовок (тип работы) |
| `state` | Москва | Город |
| `copyright` | На правах рукописи | Надпись о правах (стиль `gost`) |
| `logo` | `cmcmsuthesis-cmc` или `cmcmsuthesis-gost` | Имя файла логотипа (без расширения) |

Значение `logo` по умолчанию зависит от выбранного `titlepagesty`. Если файл логотипа не найден, титульная страница формируется без него.

### 4.2. Научный руководитель (`\supervisor{...}`)

| Ключ | Умолчание | Описание |
|---|---|---|
| `name` | уч. степень, уч. звание Фамилия Имя Отчество | ФИО и степень руководителя |
| `prefix` | Научный руководитель | Подпись перед именем |
| `prefix sep` | `: \\` | Разделитель между prefix и name |

### 4.3. Специальность (`\specialty{...}`)

Используется только в стиле `gost`.

| Ключ | Умолчание | Описание |
|---|---|---|
| `code` | X.Y.Z | Код специальности |
| `name` | название специальности... | Название специальности |
| `prefix` | Специальность | Слово перед кодом |
| `sep` | ` --- ` | Разделитель между кодом и названием |

## 5. Титульная страница

Титульная страница создаётся стандартной командой `\maketitle`.

### 5.1. Стиль `cmc` (курсовая / бакалаврская работа)

Структура сверху вниз: логотип (по центру) → университет → факультет → кафедра → «На правах рукописи» (справа) → автор → заголовок (полужирный) → подзаголовок (прописные буквы) → научный руководитель (справа) → город, год.

### 5.2. Стиль `gost` (диссертация, ГОСТ Р 7.0.11-2011)

Структура сверху вниз: университет → логотип (слева) + «На правах рукописи» (справа) → автор → заголовок (полужирный) → специальность (код --- название) → подзаголовок → научный руководитель (справа) → город --- год.

### 5.3. Логотип

Класс ищет файл логотипа по следующим путям (в порядке приоритета): текущая директория, `./images/`, `./cmcmsuthesis-latexstyles/images/`. Если пользователь задал `\graphicspath`, его пути добавляются после встроенных. Поддерживаются все форматы, распознаваемые `graphicx` (PDF, PNG, JPG и др.).

Если файл не найден, титульная страница генерируется без логотипа — ошибка компиляции не возникает.

### 5.4. Команда `\IfGraphicExists`

Вспомогательная команда для условной вставки изображений:

```latex
\IfGraphicExists{mylogo}{%
    \includegraphics[width=3cm]{mylogo}%
}{%
    Логотип не найден%
}
```

## 6. Предустановленные настройки

Класс загружает пакеты комплекта со следующими опциями:

**cmcmsugost-fontselector:**
- `font` = значение опции класса `font` (по умолчанию `modern`)
- `microtype` = `true` (или `false` в режиме `draft`)

**cmcmsu-gost:**
- `tocpageword = true` (слово «Стр.» в оглавлении)
- `sectionclearpage = true` (каждый раздел — с новой страницы)

**cmcmsugost-extra:**
- `rumathop = true` (русские операторы: tg, ctg, ⩽, ⩾)
- `thm = true` (теоремные окружения)
- `extramath = true` (команды `\vect`, `\matr` и др.)
- `listings = true` (листинги с поддержкой кириллицы)
- `biber = true` (библиография по ГОСТ)
- `alg = 2e` (алгоритмы algorithm2e)

**Цвета ссылок (hyperref):**

| Тип | Цвет (RGB) |
|---|---|
| Внутренние ссылки | тёмно-синий (0, 40, 100) |
| URL | синий (0, 0, 140) |
| Цитаты | тёмно-зелёный (0, 80, 40) |

## 7. Примеры

### 7.1. Курсовая работа (стиль `cmc`)

```latex
\documentclass[font=modern]{cmcmsuthesis}

\author{Иванов Иван Иванович}
\title{Методы машинного обучения для задачи классификации текстов}
\subtitle{Курсовая работа}

\thesissetup{
    department = {Кафедра математических методов прогнозирования}
}
\supervisor{
    name = {канд.~физ.-мат.~наук, доцент Петров~П.П.}
}

\usepackage{biblatex}
\addbibresource{refs.bib}

\begin{document}

\maketitle
\tableofcontents

\section{Введение}
Текст введения.

\section{Постановка задачи}

\begin{definition}
Классификатором называется отображение $f\colon X \to Y$.
\end{definition}

\begin{theorem}
Оптимальный байесовский классификатор минимизирует
вероятность ошибки.
\end{theorem}

\section{Алгоритм}

\begin{algorithm}[H]
    \caption{Обучение модели}\label{alg:train}
    \KwInput{выборка $\{(\vect{x}_i, y_i)\}_{i=1}^n$}
    \KwOutput{модель $\hat{f}$}
    Инициализировать параметры $\vect{w}$\;
    \For{$t = 1$ \KwTo $T$}{
        Обновить $\vect{w}$ по градиенту\;
    }
    \Return{$\hat{f}(\cdot; \vect{w})$}\;
\end{algorithm}

\section{Заключение}
Текст заключения.

\printbibliography

\end{document}
```

### 7.2. Диссертация (стиль `gost`)

```latex
\documentclass[
    parent      = extreport,
    font        = libertinus,
    titlepagesty = gost
]{cmcmsuthesis}

\author{Сидоров Алексей Михайлович}
\title{Алгоритмы устойчивого оценивания параметров
       в условиях неполных данных}
\subtitle{Диссертация на соискание учёной степени
          кандидата физико-математических наук}

\supervisor{
    name   = {д-р~физ.-мат.~наук, проф. Козлов~А.Б.},
    prefix = {Научный руководитель}
}
\specialty{
    code = {1.2.3},
    name = {теоретическая информатика, кибернетика}
}

\usepackage{biblatex}
\addbibresource{dissertation.bib}

\begin{document}

\maketitle
\tableofcontents

\chapter{Введение}
Актуальность работы обусловлена...

\chapter{Обзор литературы}
Задача оценивания параметров рассматривалась в работах~\cite{...}.

\chapter{Основные результаты}

\begin{theorem}\label{thm:main}
Пусть $\matr{A} \in \mathbb{R}^{m \times n}$.
Тогда предложенный алгоритм $\algo{RobustEstimate}$
сходится со скоростью $\compl{O}(n^{-1/2})$.
\end{theorem}

\appendix
\chapter{Доказательства вспомогательных лемм}
Доказательство леммы~1...

\printbibliography[heading=bibintoc]

\end{document}
```

## 8. Краткая справка

| Опция класса | Тип | Умолчание | Описание |
|---|---|---|---|
| `parent` | строка | `extarticle` | Базовый класс |
| `fontsize` | строка | `14pt` | Размер шрифта |
| `font` | строка | `modern` | Набор шрифтов |
| `titlepagesty` | строка | `cmc` | Стиль титульной: `cmc` или `gost` |
| `draft` | bool | `false` | Черновой режим |

| Команда | Назначение |
|---|---|
| `\thesissetup{...}` | Общие метаданные (университет, факультет, …) |
| `\supervisor{...}` | Научный руководитель (name, prefix) |
| `\specialty{...}` | Специальность (code, name) — для стиля `gost` |
| `\subtitle{...}` | Тип работы (курсовая, ВКР, диссертация) |
| `\maketitle` | Генерация титульной страницы |
| `\IfGraphicExists{file}{true}{false}` | Проверка наличия изображения |
