# cmcmsuthesis-cw.cls — Руководство пользователя

**Версия:** 0.1 от 23 марта 2026 г.

Класс `cmcmsuthesis-cw` — производный от `cmcmsuthesis`, предназначенный для оформления **курсовых работ** факультета ВМК МГУ.

---

## 1. Подключение

```latex
\documentclass{cmcmsuthesis-cw}
```

Или с опциями:

```latex
\documentclass[font=pt, draft]{cmcmsuthesis-cw}
```

---

## 2. Что фиксировано

| Параметр | Значение | Пояснение |
|---|---|---|
| `parent` | `extarticle` | Документ без глав (разделы через `\section`) |
| `titlepagesty` | `cmc` | Титульная страница в стиле ВМК |
| `copyright` | *(пусто)* | Надпись «На правах рукописи» убрана |
| `subtitle` | Курсовая работа | Подзаголовок на титульной странице |

---

## 3. Опции класса

| Опция | Тип | Умолчание | Описание |
|---|---|---|---|
| `fontsize` | строка | `14pt` | Размер основного шрифта |
| `font` | строка | `modern` | Набор шрифтов (см. документацию `cmcmsugost-fontselector`) |
| `draft` | bool | `false` | Черновой режим |

---

## 4. Полный пример

```latex
\documentclass{cmcmsuthesis-cw}

\author{Кузнецов Дмитрий Сергеевич}
\title{Численные методы решения уравнений
       в частных производных}

\thesissetup{
    department = {Кафедра вычислительной математики}
}
\supervisor{
    name = {канд.~физ.-мат.~наук, доцент Волков~Н.А.}
}

\usepackage{biblatex}
\addbibresource{refs.bib}

\begin{document}

\maketitle
\tableofcontents

\section{Введение}
В данной работе рассматриваются...

\section{Постановка задачи}

\begin{definition}
Краевой задачей называется...
\end{definition}

\section{Метод конечных разностей}

\begin{theorem}
Схема устойчива при условии $\tau / h^2 \leq 1/2$.
\end{theorem}

\section{Результаты экспериментов}
Численные эксперименты проведены для...

\section{Заключение}
В работе реализован и исследован...

\printbibliography[heading=bibintoc]

\end{document}
```
