# cmcmsuthesis-phd.cls — Руководство пользователя

**Версия:** 0.1 от 23 марта 2026 г.

Класс `cmcmsuthesis-phd` — производный от `cmcmsuthesis`, предназначенный для оформления **кандидатских диссертаций** по ГОСТ Р 7.0.11-2011.

---

## 1. Подключение

```latex
\documentclass{cmcmsuthesis-phd}
```

Или с опциями:

```latex
\documentclass[font=libertinus, draft]{cmcmsuthesis-phd}
```

---

## 2. Что фиксировано

| Параметр | Значение | Пояснение |
|---|---|---|
| `parent` | `extreport` | Документ с главами (`\chapter`) |
| `titlepagesty` | `gost` | Диссертационная титульная по ГОСТ |
| `university` | Федеральное государственное бюджетное образовательное учреждение высшего профессионального образования «Московский государственный университет имени М.В. Ломоносова» | Полное официальное название вуза |
| `subtitle` | Диссертация на соискание ученой степени кандидата физико-математических наук | Подзаголовок на титульной странице |
| `copyright` | На правах рукописи | Сохраняется (стиль `gost`) |

> **Отличие от bachelor/master.** Класс использует `extreport` (главы вместо разделов) и диссертационный стиль титульной страницы с полным юридическим названием университета, надписью «На правах рукописи» и блоком специальности.

---

## 3. Опции класса

| Опция | Тип | Умолчание | Описание |
|---|---|---|---|
| `fontsize` | строка | `14pt` | Размер основного шрифта |
| `font` | строка | `modern` | Набор шрифтов (см. документацию `cmcmsugost-fontselector`) |
| `draft` | bool | `false` | Черновой режим |

---

## 4. Метаданные

Помимо стандартных `\author` и `\title`, для диссертации необходимо задать специальность и научного руководителя:

```latex
\author{Сидоров Алексей Михайлович}
\title{Алгоритмы устойчивого оценивания параметров
       в условиях неполных данных}

\supervisor{
    name   = {д-р~физ.-мат.~наук, проф. Козлов~А.Б.},
    prefix = {Научный руководитель}
}
\specialty{
    code = {1.2.3},
    name = {теоретическая информатика, кибернетика}
}
```

Подзаголовок (`subtitle`) предустановлен, но его можно переопределить, например для докторской диссертации:

```latex
\subtitle{Диссертация на соискание ученой степени
          доктора физико-математических наук}
```

## 5. Полный пример

```latex
\documentclass[font=stix]{cmcmsuthesis-phd}

\author{Сидоров Алексей Михайлович}
\title{Алгоритмы устойчивого оценивания параметров
       в условиях неполных данных}

\thesissetup{
    department = {Кафедра математической статистики}
}
\supervisor{
    name = {д-р~физ.-мат.~наук, проф. Козлов~А.Б.}
}
\specialty{
    code = {1.2.3},
    name = {теоретическая информатика, кибернетика}
}

\usepackage{biblatex}
\addbibresource{dissertation.bib}

\begin{document}

\maketitle
\tableofcontents

\chapter{Введение}
Актуальность работы обусловлена...

\chapter{Обзор литературы}
Задача оценивания параметров рассматривалась
в работах~\cite{huber1981, hampel1986}.

\chapter{Основные результаты}

\begin{theorem}\label{thm:convergence}
Предложенный алгоритм $\algo{RobustEstimate}$
сходится со скоростью $\compl{O}(n^{-1/2})$
при условии $\|\matr{A}\| \leq C$.
\end{theorem}

\begin{proof}
Следует из леммы~\ref{lem:auxiliary}
и теоремы~\ref{thm:bound}.
\end{proof}

\chapter{Экспериментальные исследования}
Результаты численных экспериментов...

\chapter{Заключение}
В диссертации получены следующие результаты...

\appendix
\chapter{Доказательства вспомогательных лемм}
Доказательство леммы~1...

\printbibliography[heading=bibintoc]

\end{document}
```

|---|---|---|---|---|

| `cmcmsuthesis-master` | Магистерская диссертация | extarticle | cmc | Магистерская диссертация |
| **`cmcmsuthesis-phd`** | **Кандидатская диссертация** | **extreport** | **gost** | **Диссертация … кандидата ф.-м. наук** |
| `cmcmsuthesis` | Универсальный | настраивается | настраивается | настраивается |
