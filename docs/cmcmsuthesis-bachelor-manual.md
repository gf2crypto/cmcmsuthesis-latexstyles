# cmcmsuthesis-bachelor.cls — Руководство пользователя

**Версия:** 0.1 от 23 марта 2026 г.

Класс `cmcmsuthesis-bachelor` — производный от `cmcmsuthesis`, предназначенный для оформления **бакалаврских выпускных квалификационных работ** факультета ВМК МГУ. Класс фиксирует базовые настройки (родительский класс, стиль титульной страницы) и предоставляет упрощённый набор опций.

---

## 1. Подключение

```latex
\documentclass{cmcmsuthesis-bachelor}
```

Или с опциями:

```latex
\documentclass[font=libertinus, draft]{cmcmsuthesis-bachelor}
```

---

## 2. Что фиксировано

Класс автоматически передаёт в `cmcmsuthesis` следующие значения, которые пользователь **не может** изменить через опции класса:

| Параметр | Значение | Пояснение |
|---|---|---|
| `parent` | `extarticle` | Документ без глав (разделы через `\section`) |
| `titlepagesty` | `cmc` | Титульная страница в стиле ВМК |
| `copyright` | *(пусто)* | Надпись «На правах рукописи» убрана |

Все остальные настройки наследуются от `cmcmsuthesis`: оформление по ГОСТ, теоремы, алгоритмы, листинги, библиография, русские операторы и т.д.

---

## 3. Опции класса

| Опция | Тип | Умолчание | Описание |
|---|---|---|---|
| `fontsize` | строка | `14pt` | Размер основного шрифта |
| `font` | строка | `modern` | Набор шрифтов (см. документацию `cmcmsugost-fontselector`) |
| `draft` | bool | `false` | Черновой режим (без microtype, рамки вместо изображений) |

---

## 4. Метаданные

Метаданные задаются так же, как в `cmcmsuthesis`. Для бакалаврской работы типичный минимум:

```latex
\author{Иванов Иван Иванович}
\title{Применение методов глубокого обучения
       к задаче сегментации изображений}
\subtitle{Выпускная квалификационная работа бакалавра}

\thesissetup{
    department = {Кафедра математических методов прогнозирования}
}
\supervisor{
    name = {канд.~физ.-мат.~наук, доцент Петров~П.П.}
}
```

---

## 5. Полный пример

```latex
\documentclass[font=pt]{cmcmsuthesis-bachelor}

\author{Иванов Иван Иванович}
\title{Алгоритмы кластеризации графов
       в задачах анализа социальных сетей}
\subtitle{Выпускная квалификационная работа бакалавра}

\thesissetup{
    department = {Кафедра системного анализа}
}
\supervisor{
    name = {д-р~физ.-мат.~наук, проф. Сидоров~А.Б.}
}

\usepackage{biblatex}
\addbibresource{refs.bib}

\begin{document}

\maketitle
\tableofcontents

\section{Введение}
Актуальность задачи кластеризации графов...

\section{Постановка задачи}

\begin{definition}
Графом называется пара $G = (V, E)$, где...
\end{definition}

\section{Описание алгоритма}

\begin{algorithm}[H]
    \caption{Спектральная кластеризация}\label{alg:spectral}
    \KwInput{граф $G = (V, E)$, число кластеров $k$}
    \KwOutput{разбиение $\{C_1, \ldots, C_k\}$}
    Вычислить матрицу Лапласа $\matr{L}$\;
    Найти $k$ собственных векторов $\matr{L}$\;
    Применить $k$-means к строкам матрицы векторов\;
    \Return{кластеры}\;
\end{algorithm}

\section{Результаты экспериментов}
Результаты приведены в таблице~\ref{tab:results}.

\section{Заключение}
В работе предложен алгоритм...

\printbibliography[heading=bibintoc]

\end{document}
```

## 6. Связь с другими классами комплекта

| Класс | Тип работы | Базовый класс | Титульная |
|---|---|---|---|
| **`cmcmsuthesis-bachelor`** | **Бакалаврская ВКР** | **extarticle** | **cmc** |
| `cmcmsuthesis` | Универсальный (ручная настройка) | настраивается | настраивается |

Все производные классы (для магистерских, курсовых, диссертаций) устроены аналогично: фиксируют `parent`, `titlepagesty` и отдельные метаданные, а остальное наследуют от `cmcmsuthesis`.
