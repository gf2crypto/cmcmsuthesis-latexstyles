# cmcmsuthesis-doctoral.cls — Руководство пользователя

**Версия:** 0.1 от 23 марта 2026 г.

Класс `cmcmsuthesis-doctoral` — производный от `cmcmsuthesis`, предназначенный для оформления **докторских диссертаций** по ГОСТ Р 7.0.11-2011.

---

## 1. Подключение

```latex
\documentclass{cmcmsuthesis-doctoral}
```

Или с опциями:

```latex
\documentclass[font=stix, draft]{cmcmsuthesis-doctoral}
```

---

## 2. Что фиксировано

| Параметр | Значение | Пояснение |
|---|---|---|
| `parent` | `extreport` | Документ с главами (`\chapter`) |
| `titlepagesty` | `gost` | Диссертационная титульная по ГОСТ |
| `university` | Федеральное государственное бюджетное образовательное учреждение высшего профессионального образования «Московский государственный университет имени М.В. Ломоносова» | Полное официальное название вуза |
| `subtitle` | Диссертация на соискание ученой степени доктора физико-математических наук | Подзаголовок на титульной странице |
| `supervisor/prefix` | Научный консультант | Вместо «Научный руководитель» |
| `copyright` | На правах рукописи | Сохраняется (стиль `gost`) |

> **Отличие от `cmcmsuthesis-phd`.** Два изменения: подзаголовок содержит «доктора» вместо «кандидата», а подпись к научному наставнику — «Научный консультант» вместо «Научный руководитель» (в соответствии с требованиями к докторским диссертациям).

---

## 3. Опции класса

| Опция | Тип | Умолчание | Описание |
|---|---|---|---|
| `fontsize` | строка | `14pt` | Размер основного шрифта |
| `font` | строка | `modern` | Набор шрифтов (см. документацию `cmcmsugost-fontselector`) |
| `draft` | bool | `false` | Черновой режим |

---

## 4. Полный пример

```latex
\documentclass[font=ebgaramond]{cmcmsuthesis-doctoral}

\author{Петров Сергей Николаевич}
\title{Теория устойчивости стохастических систем
       с марковскими переключениями}

\thesissetup{
    department = {Кафедра теории вероятностей}
}
\supervisor{
    name = {академик РАН, д-р~физ.-мат.~наук, проф. Иванов~И.И.}
}
\specialty{
    code = {1.1.5},
    name = {математическая физика}
}

\usepackage{biblatex}
\addbibresource{dissertation.bib}

\begin{document}

\maketitle
\tableofcontents

\chapter{Введение}
Настоящая работа посвящена развитию теории...

\chapter{Обзор литературы}
Основы теории устойчивости заложены в~\cite{lyapunov1892}.

\chapter{Устойчивость линейных систем}

\begin{theorem}\label{thm:stability}
Система $\dot{\vect{x}} = \matr{A}(\xi_t)\vect{x}$
экспоненциально устойчива в среднеквадратичном тогда
и только тогда, когда...
\end{theorem}

\chapter{Нелинейные системы}
Обобщение на нелинейный случай...

\chapter{Приложения к задачам управления}
Рассмотрим задачу стабилизации...

\chapter{Заключение}
В диссертации получены следующие основные результаты...

\appendix
\chapter{Вспомогательные утверждения}
Доказательство леммы~1...

\printbibliography[heading=bibintoc]

\end{document}
```
