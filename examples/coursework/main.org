* Теоретические основы
#+name: sec:theory
** Основные определения

Пусть \( \vect{x}(t)\in\mathbb{R}^n \)~--- вектор неизвестных.

#+name: def:cauchy
#+begin_definition
/Задачей Коши/ называется задача нахождения решения
#+name: eq:ode
\begin{equation}
\frac{d\vect{x}}{dt}=\vect{f}(t,\vect{x}),\quad\vect{x}(t_0)=\vect{x}_0.
\end{equation}
#+end_definition

#+name: def:lipschitz
#+begin_definition
Функция \( \vect{f} \) удовлетворяет /условию Липшица/: \( \|\vect{f}(t,\vect{x})-\vect{f}(t,\vect{y})\|\leq L\|\vect{x}-\vect{y}\| \).
#+end_definition

#+name: thm:picard
#+begin_theorem
Если \( \vect{f} \) непрерывна и липшицева (определение\nbsp{}[[def:lipschitz]]), то задача Коши\nbsp{}\eqref{eq:ode} имеет единственное решение.
#+end_theorem

#+begin_proof
Итерации Пикара и лемма\nbsp{}[[lem:gronwall]].
#+end_proof

#+name: cor:global
#+begin_corollary
Если \( \vect{f} \) глобально липшицева, решение существует на \( [t_0,+\infty) \).
#+end_corollary

#+name: lem:gronwall
#+begin_lemma
Если \( u(t)\leq a+b\int_{t_0}^t u(s)\,ds \), то \( u(t)\leq a\,e^{b(t-t_0)} \).
#+end_lemma

#+name: prop:euler-stable
#+begin_proposition
Метод Эйлера устойчив при \( |1+h\lambda|<1 \).
#+end_proposition

#+name: rem:stiff
#+begin_remark
Для жёстких систем явные методы требуют малого шага.
#+end_remark

#+name: ex:harmonic
#+begin_example
Осциллятор \( \ddot{x}+\omega^2 x=0 \) сводится к системе: \( \dot{x}_1=x_2 \), \( \dot{x}_2=-\omega^2 x_1 \).
#+end_example

#+name: prob:pendulum
#+begin_problem
Исследовать маятник \( \ddot{\theta}+\sin\theta=0 \).
#+end_problem

* Алгоритмы и листинги

#+caption: Метод Рунге--Кутты
#+name: alg:rk4
\begin{algorithm}[H]
\KwInput{$\vect{f}$, $t_0$, $\vect{x}_0$, $h$, $N$}
\KwOutput{массив решений}
\For{$i=1$ \KwTo $N$}{
    $\vect{k}_1\gets h\,\vect{f}(t,\vect{x})$\;
    $\vect{k}_2\gets h\,\vect{f}(t+h/2,\vect{x}+\vect{k}_1/2)$\tcp*{стадия 2}
    \Comment{обновляем}
    \uIf{$\|\vect{x}\|>10^{10}$}{\Return{ошибка}\;}
    \uElseIf{$h<10^{-15}$}{\Return{предупреждение}\;}
    \Else{сохранить $(t,\vect{x})$\;}
}
\Return{массив}\;
\end{algorithm}

Ссылка: алгоритм\nbsp{}\ref{alg:rk4}.

\begin{lstlisting}[language=Python,caption={Python},label={lst:py}]
import numpy as np
def euler(f, t0, x0, h, n):
    """Метод Эйлера для решения ОДУ."""
    t, x = t0, np.array(x0, dtype=float)
    for _ in range(n):
        x = x + h * f(t, x)  # Шаг Эйлера
        t += h
    return x
print("Результат:", euler(lambda t,x: -x, 0, [1.0], 0.1, 100))
\end{lstlisting}

(*@\lstInput@*): $[a,b]$, $g$, $\varepsilon$
(*@\lstOutput@*): корень $x^*$
function Bisection($a$, $b$, $g$, $\varepsilon$)
begin
    while $b - a > \varepsilon$ do
    begin
        $c \gets (a+b)/2$
        if $g(a)\cdot g(c)<0$ then $b\gets c$
        else $a\gets c$
    end
    return $(a+b)/2$
end
